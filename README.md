# SNTP_CONF
Рабочие конфигурационные файлы chrony.conf 
В конфигурационных файлах заложена работа по одной ветке синхронизации времени, через основной сервер времени постоянно по ветке "prefer". 
При обрыве связи основного сервера времени системы с сервером времени высшего stratum-a, сервер продолжает быть главным, до поломки, 
или остановки демона chronyd.service.
